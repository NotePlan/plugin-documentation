# String Interpolation

{/* ## Overview */}
You can use standard JavaScript [string interpolation](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Template_literals) (aka "Template Literals") within `Templating` tags to combine text and variables into a string of characters.

{/* prettier-ignore */}
{/*For example, using the following in your plugin code, you can provide the referenced `fname`, `mname`, and `lname` variables

```javascript
export async function templatingInterpolation(): Promise<void> {
 try {
   const data = { fname: 'Mike', mname: 'Joseph', lname: 'Erickson' }

   const result = await NPTemplating.renderTemplate('Test (Interpolation)', data)

   Editor.insertTextAtCursor(result)
 } catch (error) {
   showError(error)
 }
}
```

{/* prettier-ignore */}
In your template, you can reference these variables.

<Callout
  type="info"
  title="INTERPOLATION"
  description="Make sure you use the standard interpolation ticks (`) around the text block which contains desired variables"
/>

```markdown
<% const fname = "Frank" -%>
<% const lname = "Winter" -%>
Full Name: <%- `${fname} ${lname}` %>
```

When your template is rendered, it will appear as

```markdown
Full Name: Frank Winter
```
