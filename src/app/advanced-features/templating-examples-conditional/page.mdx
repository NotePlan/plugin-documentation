# Conditionals

#### if logic

```markdown
<%if (1 > 0) { %>
// Do something with more than 1 recipe
<% } %>
```

#### if/else logic

```markdown
<%if (1 > 0) { %>
true
<% } else { %>
else
<% } %>
```

#### if logic (single line)

_note: the next line will only appear if the current day is Saturday_

```javascript
<%if (date.dayNumber(`${np.pivotDate}`) === 6) { %>do stuff on Saturday<% } %>
```

#### if/else logic (single line)

```javascript
<%if (1 > 0) { %>true<% } else { %>else<% } %>
```
