
# Conditionals

#### if logic

```markdown
&lt;%if (1 > 0) { &gt;%
// Do something with more than 1 recipe
&lt;% } &gt;%
```

#### if/else logic

```markdown
&lt;%if (1 > 0) { &gt;%
 true
&lt;% } else { &gt;%
 else
&lt;% } &gt;%
```
