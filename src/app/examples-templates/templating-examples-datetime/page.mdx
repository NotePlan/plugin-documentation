# Date and Time
The `Templating` DateModule and TimeModule provide a number of methods which can be used within your templates. The following example demonstrates how to use a variety of these methods.

<Callout
  type="tip"
  title="FORMATTING DATES"
  description={
    'The DateModule uses the JavaScript <code>moment</code> library internally, thus you can use any of the <a href="https://momentjs.com/docs/#/parsing/string-format/" alt="MomentJS String Formats"> Moment.js string formatting options </a> to get the exact date or time format you want.' 
  }
/>
## Template
To use this example, create a new template inside NotePlan by right-clicking "Templates" in the left sidebar and then "New Template". Then copy the template below the last blue line.

```markdown
Test (DateModule)
*Demonstrates DateModule, providing date/time utilities*

#### Templates with date and time


Date now: *&lt;%- np.date.now() &gt;%*
Date now with format: *&lt;%- np.date.now("Do MMMM YYYY") &gt;%*

Today: *&lt;%- date.today() &gt;%*
Last week: *&lt;%- np.date.now("dddd Do MMMM YYYY", -7) &gt;%*
Last week of current date: *&lt;%- np.date.now("", -7) &gt;%*
Today: *&lt;%- np.date.now("dddd Do MMMM YYYY, ddd") &gt;%*
Next week: *&lt;%- np.date.now("dddd Do MMMM YYYY", 7) &gt;%*
Today w/ format: *&lt;%- date.now("dddd MMMM Do, YYYY HH:mm A") &gt;%*

Day Number: *&lt;%- date.dayNumber() &gt;%*
Day Number w/ Specific Date: *&lt;%- date.dayNumber('2021-12-25') &gt;%*

#### Day Number Constants
Sunday: *&lt;%- date.DAY_NUMBER_SUNDAY &gt;%*
Monday: *&lt;%- date.DAY_NUMBER_MONDAY &gt;%*
Tuesday: *&lt;%- date.DAY_NUMBER_TUESDAY &gt;%*
Wednesday: *&lt;%- date.DAY_NUMBER_WEDNESDAY &gt;%*
Thursday: *&lt;%- date.DAY_NUMBER_THURSDAY &gt;%*
Friday: *&lt;%- date.DAY_NUMBER_FRIDAY &gt;%*
Saturday: *&lt;%- date.DAY_NUMBER_SATURDAY &gt;%*

Week Number: *&lt;%- date.weekNumber() &gt;%*
Week Number w/ Specific Date: *&lt;%- date.weekNumber('2021-12-25') &gt;%*

Last month: *&lt;%- np.date.now("YYYY-MM-DD", "P-1M") &gt;%*
Next year: *&lt;%- np.date.now("YYYY-MM-DD", "P1Y") &gt;%*

Date tomorrow: *&lt;%- np.date.tomorrow() &gt;%*
Date tomorrow with format: *&lt;%- np.date.tomorrow("Do MMMM YYYY") &gt;%*

Date yesterday: *&lt;%- np.date.yesterday() &gt;%*
Date yesterday with format: *&lt;%- np.date.yesterday("Do MMMM YYYY") &gt;%*

Add 3 Business Days: *&lt;%- np.date.businessAdd(3) &gt;%*
Add 3 Business Days w/ PivotDate 2021-11-24: *&lt;%- np.date.businessAdd(3,'2021-11-24') &gt;%*

Subtract 3 Business Days: *&lt;%- np.date.businessSubtract(3) &gt;%*
Subtract 3 Business Days w/ PivotDate 2021-11-22: *&lt;%- np.date.businessSubtract(3,'2021-11-22') &gt;%*

This week's Monday: *&lt;%- np.date.weekday("YYYY-MM-DD", 1) &gt;%*
Next Monday: *&lt;%- np.date.weekday("YYYY-MM-DD", 8) &gt;%*

Week Of: *&lt;%- np.date.weekOf() &gt;%*
Week Of (only pivotDate): *&lt;%- np.date.weekOf('2021-11-03') &gt;%*
Week Of: *&lt;%- np.date.weekOf(null, null, '2021-11-03') &gt;%*

Current Time: *&lt;%- np.time.now('h:mm A') &gt;%*
Current Date: *&lt;%- date.now() &gt;%*
```
