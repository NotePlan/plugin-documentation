- /swt [try it](noteplan://x-callback-url/runPlugin?pluginID=nmn.sweep&command=swt)

### Commands
- /Tidy Up [try it](noteplan://x-callback-url/runPlugin?pluginID=np.Tidy&command=Tidy%20Up)
  *Run as many of the other commands in this plugin as you have configured.*
- /File root-level notes [try it](noteplan://x-callback-url/runPlugin?pluginID=np.Tidy&command=File%20root-level%20notes)
  *For each root-level note, asks which folder you'd like it moved to. (There's a setting for ones to ignore.)*
- /List conflicted notes [try it](noteplan://x-callback-url/runPlugin?pluginID=np.Tidy&command=List%20conflicted%20notes)
  *Creates/updates a note that lists all notes on this device that have conflicts. (Requires NP v3.9.3)*
- /List duplicate notes [try it](noteplan://x-callback-url/runPlugin?pluginID=np.Tidy&command=List%20duplicate%20notes)
  *Creates/updates a note that lists potentially duplicate notes because they have identical titles*
- /List stubs [try it](noteplan://x-callback-url/runPlugin?pluginID=np.Tidy&command=List%20stubs)
  *Creates/updates a note that lists all your notes that have note links (wikilinks) that lead nowhere*
- /Remove @done() markers [try it](noteplan://x-callback-url/runPlugin?pluginID=np.Tidy&command=Remove%20%40done%28%29%20markers)
  *Remove @done() markers from recently-updated notes. Can be used with parameters from Template or Callback.*
- /Remove blank notes [try it](noteplan://x-callback-url/runPlugin?pluginID=np.Tidy&command=Remove%20blank%20notes)
  *Remove notes that are completely empty, or just have a `#` character*
- /Remove orphaned blockIDs [try it](noteplan://x-callback-url/runPlugin?pluginID=np.Tidy&command=Remove%20orphaned%20blockIDs)
  *Remove blockIDs from lines that had been sync'd, but are 'orphans' as the other copies of the blockID have since been deleted.*
- /Remove time parts from @done() dates [try it](noteplan://x-callback-url/runPlugin?pluginID=np.Tidy&command=Remove%20time%20parts%20from%20%40done%28%29%20dates)
  *Remove time parts of @done(date time) from recently-updated notes. Can be used with parameters from Template or Callback.*
- /Remove section from recent notes [try it](noteplan://x-callback-url/runPlugin?pluginID=np.Tidy&command=Remove%20section%20from%20recent%20notes)
  *Remove a given section (both the heading and its content) from recently-changed notes.
Can be used with parameters from Template or x-callback.*
- /Remove section from all notes [try it](noteplan://x-callback-url/runPlugin?pluginID=np.Tidy&command=Remove%20section%20from%20all%20notes)
  *Remove a given section (both the heading and its content) from all notes.
Can be used with parameters from Template or x-callback.*
- /Remove triggers from recent calendar notes [try it](noteplan://x-callback-url/runPlugin?pluginID=np.Tidy&command=Remove%20triggers%20from%20recent%20calendar%20notes)
  *Remove one or more triggers from recent (but past) calendar notes.
Can be used with parameters from Template or x-callback.*
- /Log notes changed in interval (Tidy) [try it](noteplan://x-callback-url/runPlugin?pluginID=np.Tidy&command=Log%20notes%20changed%20in%20interval%20%28Tidy%29)
  *Write a list of Log notes changed in the last interval of days to the plugin log. It will default to the 'Default Recent Time Interval' setting unless passed as a parameter.*
